Scheme Interpreter
Racket

A Scheme interpreter using Continuation-Passing Style (CPS) and recursion evaluates expressions by explicitly passing control through continuation functions. Instead of returning values directly, each function calls its continuation with the result, allowing for non-standard control flows like tail-call optimization and first-class continuations. The interpreter recursively processes expressions, handling variables, functions, conditionals, and macros while ensuring efficient execution without stack overflow issues.

Dependency Health Visualizer
R, TypeScript, Python

The Dependency Health Visualizer is a tool designed to analyze and visualize Node.js dependency trees, helping developers identify vulnerable or unreliable packages. It assigns health scores to dependencies using a machine learning regression model trained on key package attributes. The tool represents dependencies in an interactive, color-coded tree graph, making it easy to assess package relationships and risks at a glance. It features subtree navigation, scalable visualization, and intuitive interaction, allowing users to explore dependencies without information overload. The system processes npm package data, extracts relevant information through web scraping, and structures it into a tree format.

Editor Trees
Java (Algorithms)

The project entails writing methods for a class representing the underlying data structure of a text editor. Each piece of text is represented as a height-balanced tree with rank and balance codes, where each node contains a single character. This structure, known as an EditTree, focuses on the efficient manipulation of text rather than alphabetical ordering. The emphasis is on implementing complex data structures and associated algorithms, utilizing debugging tools effectively, interpreting algorithm descriptions for Java implementation, and practicing teamwork skills.
